# bee.tinper.org

bee组件库文档官网。

[tinper-bee](http://github.com/iuap-design/tinper-bee.org)

[组件仓库](http://github.com/tinper-bee)

组件以git子模块的模式存放在tinper-bee文件夹内。

### 调试

- clone代码后，执行`npm run init`下载子模块

```
npm run init
```

- 更新子模块

```
npm run update
```

- 启动服务（要查看文档完整性，个别文档会有不显示和显示错误）

```
npm run server
```

会监听9090端口。
